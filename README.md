# LibreTV

一个轻量级、自托管的影视聚合与个人媒体平台，支持 WebGL 实时超分辨率和 Cloudflare R2 个人网盘功能。

## 核心贡献

基于 bestK/tv 进行 637+ 次迭代，主要技术增强如下：

- **WebGL 实时超分辨率**：在播放器中集成 Anime4K、CAS 以及针对老剧优化的两遍超分方案，实现客户端实时画质增强，无需服务端额外算力。
- **个人媒体库（R2 网盘模式）**：将 Cloudflare R2 作为个人网盘使用，支持浏览器直传（S3 兼容 SigV4 预签名）、基于用户前缀的严格权限隔离，以及支持 Range 请求的私有串流代理。
- **多用户账号系统 + KV 云同步**：使用 Cloudflare Workers KV 存储用户数据（历史、收藏、设置按用户隔离），并将用户密码通过 PBKDF2 + 加盐方式安全存储在 KV 中，实现真正的多用户私有云体验。
- 中英文双语切换、液态玻璃界面、多源智能聚合与去重。

在线演示：https://lyj111.dpdns.org

## 技术亮点

### WebGL 实时超分辨率
在播放器中基于 WebGL2 实现实时画质增强，支持：
- Anime4K（动画内容）
- 对比度自适应锐化（CAS，适合实拍内容）
- 老剧专用两遍超分（先降噪再锐化）
- 支持高性能超采样与实时强度调节

该方案完全在客户端 GPU 完成，解决了低质量源在 Web 端画质差的问题，同时避免了服务端渲染带来的额外成本和延迟。

### R2 个人媒体库（网盘模式）
将 Cloudflare R2 作为私有网盘使用：
- 浏览器直传 R2（绕过 Worker 请求体限制）
- 基于用户前缀的严格权限隔离，防止越权访问
- 支持 Range 请求的私有串流代理，可拖动播放
- 与多用户账号系统深度整合

### 多用户系统与 KV 数据隔离
将用户密码通过 **PBKDF2 + 每用户随机盐** 安全存储在 Cloudflare KV 中，同时实现：
- 用户历史、收藏、设置的按账号云端隔离与同步
- 会话使用 HMAC 签名 + HttpOnly Cookie 管理
- 优雅降级：未登录时完全维持原有本地体验

该设计在保持轻量部署的同时，提供了比单一站点密码更安全、更灵活的多用户能力。

## 技术栈

- 前端：原生 HTML + Tailwind CSS + ArtPlayer + WebGL2
- 后端：Cloudflare Pages Functions
- 存储：Cloudflare Workers KV + R2
- 鉴权：Web Crypto（HMAC + PBKDF2）
