# LibreTV

一个轻量级、自托管的影视聚合与个人媒体平台，支持 WebGL 实时超分辨率和 Cloudflare R2 个人网盘功能。

## 核心贡献

基于 bestK/tv 进行 637+ 次迭代，主要完成了以下技术增强：

- **WebGL 实时超分辨率**：在播放器中集成 Anime4K、CAS 以及针对老剧优化的两遍超分方案，支持实时画质增强与强度调节。
- **个人媒体库（R2 网盘）**：将 Cloudflare R2 作为个人网盘使用，支持浏览器直传（S3 兼容 SigV4 预签名）、权限隔离、私有 Range 串流播放，单文件支持 ≤5GB。
- 多用户账号系统 + KV 云同步（历史、收藏、设置按用户隔离）。
- 中英文双语切换、液态玻璃界面、多源智能聚合与去重。

在线演示：https://lyj111.dpdns.org

## 技术亮点

### 1. WebGL 实时超分辨率
在播放器中实现基于 WebGL2 的实时画质增强，支持：
- Anime4K（动画内容）
- 对比度自适应锐化（CAS，适合实拍内容）
- 老剧专用两遍超分（先降噪再锐化）
- 支持高性能超采样与实时强度调节

这部分解决了低质量源在 Web 端播放画质差的问题，且完全在客户端 GPU 完成，无需额外服务端算力。

### 2. R2 个人媒体库（网盘模式）
将 Cloudflare R2 作为私有网盘使用：
- 浏览器直传 R2（绕过 Worker 请求体限制）
- 基于用户前缀的严格权限隔离
- 支持 Range 请求的私有串流代理，可拖动播放
- 与多用户账号系统深度整合

这个设计让用户可以在站内像使用个人云盘一样上传和管理自己的视频内容。

## 技术栈

- 前端：原生 HTML + Tailwind CSS + ArtPlayer + WebGL2
- 后端：Cloudflare Pages Functions
- 存储：Cloudflare Workers KV + R2
- 鉴权：Web Crypto（HMAC + PBKDF2）
